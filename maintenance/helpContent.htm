Now let's talk about the organizational hierarchy.

Pages

Pages are your basic content.

There is a special editor for providing the page content.  In addition to the general textual content pages also own pictures and downloadable documents (such as PDF files). 

The editor provides for three methods of representing the textual content:

  • Simple Text Format -- This is a very simple text formatting scheme.  Simple things like paragraphs and bullet lists are very simple to do.  More complex things like inserting pictures are a bit more involved but still simple (and much less complex than HTML).
  • Forum Text Format -- This formatting scheme follows the same as many web-based forums.  Those people that are already comfortable with posting to forums may want to use the text entry method.
  • Hyper-Text Format -- While HTML can be complex, we provide a simple visual editor that allows you to enter the text naturally.  At this point placing pictures through the HTML editor is not supported but will be addressed at some point in the future.
  • Plain Text -- We suggest not using the Plain text method of entry because it rarely provides a good rendering, but we have provided it for completeness.

Categories

Each page belongs to a single category.  In the page editor there is a drop-down that identifies to which category the page belongs.

Tabs

One or more categories can be associated to a tab.  A single category can be associated to more than one tab.  All of the pages that are associated to the categories for a tab are sorted and listed in the menu.  The tab defines the sort-order and presentation for the pages.

Home Tab

The Home Tab is special because it displays "pages" and other content in unique ways.

We have a special editor that let's you identify what you want displayed on the Home Tab and where you want it displayed.
